B1：排序类
S01-S10分别是冒泡排序、快速排序、插入排序、希尔排序、选择排序、堆排序、归并排序、计数排序、桶排序、基数排序

B2：数组类
CombinationSum 和为指定数的所有组合：回溯 动态规划
FourSum 四数之和：排序+回溯剪枝
LengthOfLIS 最长上升子序列：动态规划
SortColors 三色排序：双指针
Subsequence 数组子序列：回溯
SubsequenceExpectation 连续子序列最大值的期望：动态规划
ThreeSumClosest 最接近的三数之和：排序+双指针
TopKFrequent 前K个高频元素：优先队列（最小堆）

B3：字符串类
CreatePalindrome 构造回文的最少删除次数：动态规划，同LengthOfLCS
LengthOfLCS 最长公共子序列：动态规划 （和最小编辑距离类似）
LongestPalindrome 最长回文子串：动态规划或中心扩散
LongestSubstring 至少K个字符的最长子串：分治法+DFS
MinEditDistance 最小编辑距离：动态规划（和最长公共子序列类似）
MinWindow 最小覆盖子串：滑动窗口
ReorganizeString 重构字符串使相邻字符不同：基于计数的贪心算法

B4：链表类
LRUCache LRU缓存：双链表应用
MergeKLists 合并K个排序链表：优先队列
MergeTwoLists 合并2个排序链表
ReverseKGroup K个一组反转链表：递归
ReverseList 反转链表：递归或迭代

B5：树类
InvertTree 翻转二叉树：BFS
LowestCommonAncestor BST的最近公共祖先

B6：图类
FindDisjointCircle 寻找冗余连接：并查集
MinCost 遍历无向加权图的最短路径：排序+并查集
NumIslands 0-1矩阵岛屿数量：BFS或DFS
NumIslands2 无向图岛屿数量：BFS或DFS或并查集
UpdateMatrix 01矩阵：DP或DFS或BFS
ShortestPath 公交换乘：BFS最短路径

B7：双端队列类
MonotonousQueue 滑动窗口的最大值：单调队列
MonotonousStack 下一个最大元素、直线高楼问题：单调栈

B8：动态规划类
PackSack 背包问题：动态规划
